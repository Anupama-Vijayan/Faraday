
The input file specifications for Question 1 are:

1. The first line contains an integer n, denoting the number of elements to be there in the list when it is initialized.
2. Each line i of the n subsequent lines contains elements with which the list is initialized.
3. The subsequent lines, contains one of the commands ( along with their arguments) as described:
	unique : to find unique items from the list
	frequency : to find all items and their frequency
	insert :  to insert new items to the list
	append: to append new items to the list
	print: to print the list
